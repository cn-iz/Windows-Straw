# i-Windows-c-wpf-
利用 c# wpf 开发一个Windows平台获取电脑屏幕鼠标位置的颜色代码
<br/>
<h2>项目背景：</h2>
<br/>
2018年春节太无聊，想学学Windows桌面开发，于是做了i取色器和z录屏
<br/>
<br/>
1.项目vs2017基于
2.i取色器目录为打包项目生成可安装程序
3.取色器目录为项目文件
<br/>
<br/>
<h2>bug说明：</h2>
<br/>
在后来实际使用中发现当某个通道值小于16（转为16进制只有一位）时转换为16进制的算法有点问题，即#00ffff会换算为#0ffff。
由于没有安装vs2017一直没有修改。
